# Contador de usos de palavras
Um console app em C#que navega por arquivos texto (como livros ou artigos por exemplo) e descobre a quantidade de usos de cada palavra existente no dicionário disponibilizado.

## Arquivos e pastas
- O arquivo dicionário fonte contendo todas as palavras com 5 letras deve ficar na raiz e se chamar `dic.txt`
- Arquivos texto (artigos, livros, etc) devem ficar na pasta `/uses/*.txt`
- O arquivo de dicionário processado com as contagens é exportado na pasta raiz como `result-dic.json`
